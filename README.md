## ByBike - Pojeto para logistica de bikes

# Frameworks
| Nome | Descrição | Link |
| ------:| -----------:| -----------:|
| fastify   | Micro framework para requisições http. Alternativa ao express | [fastify.io](https://www.fastify.io/) |
